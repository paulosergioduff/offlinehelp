                Baixo e alto nível diz respeito a proximidade da máquina.
                Por exemplo, Assembly é baixo nível pois você está usando a linguagem do processador para programar, apesar de você usar mnemônicos (palavras que representam as instruções de máquina), elas tem uma relação de 1:1, isto significa que uma instrução em Assembly como MOV é apenas um apelido para o Opcode 88(hexadecimal) ou 10001000(binário) que é a instrução do Intel 80386 que move um byte do registrador para outro registrador ou segmento de memória.
                No caso das linguagens de alto nível, um único comando pode representar muitas e muitas linhas na linguagem de baixo nível.
