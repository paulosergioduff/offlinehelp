Como fazer notificações Push com o Ionic 2 + OneSignal

Use o recurso de Push Notifications para melhorar a comunicação e o engajamento do seu aplicativo com os usuários. Por que o OneSignal? Porque é grátis e ilimitado!

Crie um projeto Ionic 2. Se você não sabe como fazer isso: clique aqui. Você vai precisar criar uma conta no OneSignal. Se ainda não tem, clique aqui.

No seu painel do OneSignal, clique em “Add new app” para criar um novo aplicativo a ser vinculado com o seu projeto Ionic 2. Particularmente para esse projeto, usaremos o nome “tutorial-ionic”.

Depois de adicionar um aplicativo no OneSignal, no painel, você vai precisar configurar as plataformas para as quais as notificações serão enviadas. Clique em “App Settings” no menu lateral esquerdo e vá para a aba “Platforms”.

Nesse tutorial, vamos habilitar as notificações no Google Android. Para isso, vamos precisar gerar uma chave de API de servidor (server API key) do Google.

Abra o seu Console Firebase e faça login com sua conta do Google (sim, para o Android você precisa de uma).

Você vai precisar de um projeto no Firebase. Se você não sabe como criar um, clique aqui. No painel do seu projeto, no canto superior esquerdo, clique no ícone de engrenagem e em “Configurações do projeto”.

Nas configurações, vá para a aba “Cloud Messaging” e guarde os códigos de “Server Key” e “Código do remetente”, destacados na Figura 4:

Volte à tela do painel do seu aplicativo, no OneSignal, e clique no botão “Configure” da plataforma Google Android.

Na janela que será aberta (Figura 6), no campo “Google Server API Key”, insira o valor da “Server key” que você guardou anteriormente, e no campo “Google Project Number”, insira o valor do “Código do remetente”.

Vamos agora executar uma instrução, via linha de comando, dentro do diretório do nosso projeto Ionic 2. Se você não sabe fazer isso, clique aqui.

ionic plugin add onesignal-cordova-plugin

Depois execute:

npm install --save @ionic-native/onesignal

No arquivo app.components.ts do seu projeto Ionic 2, altere o método initializeApp :

initializeApp() {
   this.platform.ready().then(() => {
      let funcaoRetorno = (data) => {
         console.log('Notificações: ' + JSON.stringify(data));
      };

      window["plugins"].OneSignal.startInit("SUA_APPID",
          "NUMERO_PROJETO_GOOGLE")
          .handleNotificationOpened(funcaoRetorno)
          .endInit();
  });
}

Substitua o marcador SUA_APPID do trecho de código acima com a sua chave de aplicação do OneSignal, que pode ser encontrada na aba “Keys & IDs” do menu lateral esquerdo “App Settings”.

No marcador NUMERO_PROJETO_GOOGLE , substitua “Google Project Number” que você gerou anteriormente.

Volte ao painel inicial do OneSignal e clique no seu projeto. Você precisa agora selecionar o SDK que será usado pelo OneSignal e fazer a primeira inscrição para completar os passos. Como estamos usando o Ionic 2, na seleção do SDK, use a opção Ionic, ilustrada na Figura 9. Depois clique em “Next”.

Hora de testar no aplicativo, via celular Android!

Execute o comando do Ionic CLI para gerar uma apk e instalar no seu celular para testes. Uma vez que o OneSignal foi configurado no seu projeto para fazer e receber notificações usando o cordova, os testes devem ser feitos no celular (ionic serve + browser não vai funcionar).

  1. Adicione a plataforma Android ao seu projeto Ionic 2:
ionic platform add android

2. Para se certificar que o plugin foi adicionado ao seu projeto, execute também:

npm install

3. Gere o arquivo apk para ser instalado no seu celular android:

ionic build android --prod --aot

4. Conecte seu celular ao computador em que está o projeto Ionic 2 e transfira para ele o arquivo com extensão .apk que se encontra na pasta:

SEU_PROJETO/platforms/android/build/outputs/apk

5. Depois de transferir, instale o aplicativo no seu celular (você pode precisar de permissões especiais para isso) e deixe-o aberto por enquanto.

Enviando uma notificação push

Volte ao painel do OneSignal e clique no seu projeto. Depois, clique em “New push notification”

Escolha a opção “Send to Everyone” (enviar para todo mundo). e depois em “Next”.

Escolha um título (campo title) e uma mensagem (campo message) para criar a notificação. Depois clique em “Next”.

Nesse tutorial, vamos usar as configurações padrão de notificação do OneSignal  — você pode configurá-las se quiser. Aqui são definidos parâmetros como cores, sons, ícones e mais. Para ganhar tempo, não vamos mudar nada na tela da Figura 16. Clique em “Next”.

Você pode escolher se a notificação será enviada agora ou ser agendada para uma data e horários específico. Aqui, vamos enviar automaticamente, deixando selecionadas as opções “Begin sending immediately” e “Send Immediately”.

Na tela de resumo dos dados e configurações, clique em “Send message”.

Pronto! Se você seguiu corretamente os passos, seu celular acaba de receber uma notificação push. No meu celular, ficou assim:

window.plugins.OneSignal.getIds(ids => {
  var msg = { 
      contents: {
        en: "message body"
