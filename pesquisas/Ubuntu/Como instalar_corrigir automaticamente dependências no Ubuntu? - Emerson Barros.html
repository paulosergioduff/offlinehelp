Como instalar/corrigir automaticamente dependências no Ubuntu?

Postado em: 6 de setembro de 2014 por: Emerson Barros — 3 Comentários ↓

Se ao instalar um pacote dpkg -i pacote.deb no Ubuntu (ou derivados) você a mensagem de erro mostrando que é necessário instalar dependências, basta executar o comando abaixo:

sudo apt-get -f install

Fonte: http://askubuntu.com/questions/40011/how-to-let-dpkg-i-install-dependencies-for-me
