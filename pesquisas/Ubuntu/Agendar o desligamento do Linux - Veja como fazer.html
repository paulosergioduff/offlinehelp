Como agendar o desligamento do Linux

Precisa agendar o desligamento do Linux, para que seu PC seja desligado em uma determinada hora? Conheça duas formas bem práticas e simples de fazer isso!

    Se você precisa agendar o desligamento do Linux, para que seu PC seja desligado em uma determinada hora, conheça a seguir duas formas bem práticas e simples de fazer isso.

    • Programando o Linux para desligar automaticamente com Gshutdown
    • Como remover a caixa de diálogo de confirmação de desligamento no Ubuntu
    • Como agendar o desligamento do Linux

    As vezes você pode precisar “programar” o Linux para ser desligado depois de algum tempo, e nessa hora, você pode fazer isso usando a linha de comando ou uma ferramenta gráfica. Mas não se preocupe, pois abaixo mostrarei como fazer isso das duas maneiras.

    Como agendar o desligamento do Linux pela linha de comando (terminal)

    Agendar o desligamento do Linux pela linha de comando é muito fácil. Tudo que você tem a fazer é abrir um terminal e usar o comando “shutdown”.

    Skip

    Por exemplo, para marcar uma paralisação às 03:30 AM, você pode usar o comando na seguinte forma:

    sudo shutdown -h 03:30

    Note que para funcionar, você terá que manter a execução desse comando, e para isso, será preciso manter o terminal aberto.

    Como agendar o desligamento do Linux com o EasyShutdown

    EasyShutdown é um aplicativo que fornece uma pequena interface gráfica onde você pode facilmente agendar um desligamento no Linux.
