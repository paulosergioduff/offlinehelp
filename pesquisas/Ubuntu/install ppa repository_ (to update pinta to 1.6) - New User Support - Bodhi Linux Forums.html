          Hi Community,

          I'm trying bodhi for some days now and I love it! I'm afraid that I have a lot of questions though...

           

          Problem:

          The Pinta (image editor) version available in the appcenter/synaptic package manager is the old 1.3 version.

          It has a known bug which makes this version unuseable. This got fixed in the 1.6 version. (Selecting and cropping does not work) More about this bug HERE

           

          Unfortunately I cannot update the 1.3 version.

          Pinta recommends to use a more lately ppa's. 

          available here

           

          When I try add-apt-repository:

           

          $ sudo add-apt-repository ppa:pinta-maintainers/pinta-stable
          [sudo] password for bodhi: 
          sudo: add-apt-repository: command not found
           
          It says command not found.
          Can you help me here? (I know using ppa's is not recommended but I don't know what to do. I'd love to use Pinta.)

          If you don't want to wait for it to be added and are willing to use ppa's then enter the following into terminal:

          sudo apt-get install software-properties-common

          Then follow your instructions for adding ppa and installing the software

          Just posting my normal word of warning that PPAs are untested software that can make your computer unstable and difficult to support. This isn't just true for Bodhi it is true for any Ubuntu based system you are adding them to.

           

          Making a software request is ideal so the Bodhi team can test the software.

           

          I've added Pinta 1.5 to the Bodhi repos. According to their website that is the latest stable version as of today.
