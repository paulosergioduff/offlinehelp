Depende do host, mas você provavelmente simplesmente não pode (você não pode no meu host compartilhado em Rackspace Cloud Sites – eu perguntei).

O que você pode fazer é configurar um ambiente em sua máquina do dev que corresponda grosso modo ao seu host compartilhado e faça todo seu gerenciamento através da linha de comando localmente. Então, quando tudo estiver configurado (você puxou todas as dependencies, atualizadas, gerenciadas com git, etc.), você pode “empurrar” para o seu host compartilhado sobre o (s) FTP.

Este tutorial funcionou para mim, resolvendo meus problemas com / usr / local / bin permissão e php-cli (que o compositor exige, e pode ser diferente em hospedagem compartilhada).

Execute primeiro esses comandos para baixar e instalar o compositor: 

 cd ~ mkdir bin mkdir bin/composer curl -sS https://getcomposer.org/installer | php mv composer.phar bin/composer

Determine a localização do seu php-cli (necessário mais tarde): 

 which php-cli

(Se o acima falhar, use o which php )

Ele deve retornar o caminho, como / usr / bin / php-cli, / usr / php / 54 / usr / bin / php-cli, etc.

edite ~ / .bashrc e verifique se esta linha está no topo, adicionando-o se não for: 

 [ -z "$PS1" ] && return

e, em seguida, adicione este alias à parte inferior (usando o caminho php-cli que você determinou anteriormente): 

 alias composer="/usr/bin/php-cli ~/bin/composer/composer.phar"

Conclua com estes comandos: 

 source ~/.bashrc composer --version

Eu instalei o Composer (e Laravel) com sucesso na minha hospedagem compartilhada com apenas access FTP:

  1. Baixe e instale o PHPShell em um host compartilhado

  2. No config.php da PHPShell, adicione um usuário e um alias:

    php = "php -d suhosin.executor.include.whitelist=phar"

  3. Faça o login em PHPShell e digite: curl -sS https://getcomposer.org/installer | php curl -sS https://getcomposer.org/installer | php

  4. Quando instalado com sucesso, execute Composer: php composer.phar

Você pode fazê-lo dessa maneira:

  • Crie um diretório onde você deseja instalar o compositor (digamos / home / your_username / composer)
  • Vá para este diretório – cd /home/your_username/composer

  • Em seguida, execute o seguinte comando:

    php -r "readfile('https://getcomposer.org/installer');" | php

  • Depois disso, se você quiser executar o compositor, você pode fazê-lo dessa maneira (neste caso, você deve estar no diretório do compositor): php composer.phar

  • No próximo passo, você pode fazer isso:

    alias composer="/home/your_username/composer/composer.phar" .

    E execute comandos como o faça normalmente: $ composer install
