                    Citação
                  aí pessoal to com uma porcaria de um programa com a extensão .exe eu quero ver os códigos fonte dele mas não sei como fazer c vocês puderem me ajudar estarei grato!!!

                  Se você fizer um programa, em C por exemplo, e compilar, o resultado será um arquivo .exe. O código fonte é esse arquivo escrito, e o executável são instruções que o computador vai seguir. Portanto, você precisa desse arquivo editado com o compilador, e não do executável, pra ter acesso ao código fonte.

                  E se você abrir o executável com o compilador, vai aparecer um montão de caracteres malucos e não o código fonte.

                  P.S. O programa não é só o executável(.exe), envolve muito mais coisas, dependendo da complexidade do programa.

                    Citação
                  sei q programas como .vbs e .bat da para editalos no wordpad mas como faço isso com os exe?

                  Arquivos executáveis não podem ser editados, é impossível. O que é editado é o código fonte, e a partir da edição um novo executável precisa ser gerado.
