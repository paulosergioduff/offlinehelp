                      Excelente post!

                      Fiz uma instalação nova com Nginx. O usuário/grupo da pasta html (/var/www/html) é meu usuário local e seguindo a mesma lógica do post, fiz o seguinte:

                      Alterei o usuário do Nginx para meu usuário local no arquivo nginx.conf (primeira linha do arquivo) :

                      sudo nano /etc/nginx/nginx.conf

                      Reiniciei o nginx:

                      sudo systemctl restart nginx

                      Alterei também o usuário/grupo no arquivo www.conf do PHP, que no meu caso fica em /etc/php/7.0/fpm/pool.d/www.conf:

                      sudo nano /etc/php/7.0/fpm/pool.d/www.conf

                      Linhas alteradas (mudar o www-data para seu usuário Linux):
                      user = www-data
                      group = www-data

                      listen.owner = www-data
                      listen.group = www-data
                      listen.mode = 0660 (descomentar a linha retirando o ";" do início)

                      Reiniciei o PHP:

                      sudo systemctl restart php7.0-fpm
